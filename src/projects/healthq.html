HealthQ is my capstone project from my senior year at Kennesaw State University. This is a project sourced from the U.S. Centers for Disease Control and Prevention to build an artificial intelligence component that summarizes any given outbreak data set on demand and perform the relevant analyses to determine the most likely cause of outbreak.

Technologies

In the time frame of one semester, and using an Agile methodology, our software development team developed a prototype that will serve as the foundation for future work on this project.

Currently, HealthQ supports the following features:

  • Voice data input & data retrieval
  • Drawing bulk data from databases
  • Understands some CDC jargon and in-house SODA API
  • Basic statistical analysis capabilities
  • Editable modules for different implementations
