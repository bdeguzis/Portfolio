MamoBooks is a web-based accounting application. This project was created during a class at Kennesaw State University titled, “Application Domain” wherein we were were presented with a domain completely new to us (finance) by our client who is an expert in the field. We were tasked with creating a web application for our client while simultaneously learning the necessary domain knowledge.

Features

MamoBooks has the following features:

  • Creating and managing various user accounts
  • Creating, viewing and editing a chart of accounts
  • Journalizing and viewing journalized transactions
  • Reviewing, accepting, and rejecting pending journal entries.
  • Posting transactions to accounts
  • Creating and viewing trial balances
  • Creating and viewing various financial statements
    • Income statements
    • Balance sheets
    • Cash flow statements
    • Statement of retained earnings
  • Viewing a dashboard that shows various statistics and financial ratios (ROA, ROE, NPM, etc).
  • Searching and filtering sitewide datasets.
  • Viewing a detailed log of all actions done by any and all users.
