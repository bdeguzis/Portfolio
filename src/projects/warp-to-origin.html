About

Warp To Origin is a space themed shoot 'em up game for the PlayStation Vita developed in C# using the PlayStation Mobile framework. The name, “Warp to Origin” is an homage to my very first large scale software project. I am now picking up where I left off when I first starting coding.

Checkout the GitHub page here.

Features

This game features:

  • Face paced shoot 'em up game play
  • Multiple 'Bullet Hell' style enemies
  • Various weapon upgrades
  • Score Attack Mode
  • Leaderboards
  • And More!

Warp 2 Origin

Due to Sony cutting support for the PlayStation Mobile framework that this game was built on, I have decided to port it to Unity, a more modern and versatile engine. This will allow me to expand the audience of this game to Windows, Mac, and Linux users, as well as continue development for the PlayStation Vita. Look out for future releases on all these platforms coming later in 2019!
